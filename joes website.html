Camp for Coders 2015

Improve your general software literacy as we create interactive and visual computational artifacts. Learn the syntax and concepts of software (variables, functions, object oriented programming). We will write short prototypical pieces of software that explore photography, drawing, animation, and presentation.

Processing 2D

June 1 - 11, Mon - Thur

Sketch and animate 2D graphics in the Processing language. An introduction or review of programming fundamentals: variables, functions, and object oriented design.

Completed

Pixel Processing

June 15 - 18

Explore programmatic manipulation of images pixel by pixel. Apply photo effects: blending, transformation, tint, transparency, opacity, hue, saturation, brightness, erosion, dilation, posterization, greyscale, color inversion.

Completed

Web Design

June 22 - 25

Learn or review the basics of web design in this crash course on HTML, CSS, and Javascript. Regard the challenge of designing content for a variety of devices and platforms in what is commonly referred to as responsive web design.

Begin
